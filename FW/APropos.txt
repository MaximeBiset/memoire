FRENCH :
========

Cette application web a été développé fin 2014 par un groupe d'étudiants (Denis Genon, Thibault Gerondal, Michaël Heraly, Baptiste Lacasse, Arnold Moyaux, Aloïs Paulus, Jérémy Vanhee et Victor Velghe) de l'Ecole Polytechnique de Louvain à l’Université catholique de Louvain, dans le cadre du cours Software Development Project, enseigné et encadré par le professeur Kim Mens (kim.mens@uclouvain.be) et son équipe didactique. Ce projet a été réalisé en collaboration avec Het BuurtPensioen (in French “Pens(i)onsVoisins”), un projet du Kenniscentrum Woonzorg Brussel,  en particulier grâce aux conseils de Katleen Deruytter.  De plus, cette application sera utilisée comme point de départ pour le développement d'un framework, dans le cadre d'un mémoire suivi par le professeur Kim Mens, destiné à créer un logiciel applicable à plusieurs organisations du même type. L'entièreté de ce projet et son extension éventuelle est sous licence AGPv3,  dont le texte intégral peut être trouvé ici : https://www.gnu.org/licenses/agpl-3.0.html.  Le code source peut être téléchargé ici : https://github.com/MaximeBiset/care4care. En cas de problème, merci de contacter maxime.biset@student.uclouvain.be.


ENGLISH :
=========

This web application was developed at the end of 2014 by a group of students (Denis Genon, Thibault Gerondal, Michaël Heraly, Baptiste Lacasse, Arnold Moyaux, Aloïs Paulus, Jérémy Vanhee et Victor Velghe) of the Louvain School of Engineering at the Université catholique de Louvain, in the context of the course Software Development Project, taught and organised by professor Kim Mens (kim.mens@uclouvain.be) and his assistants. The project was realised in collaboration with Het BuurtPensioen (in English “Care4Care”), a project of Kenniscentrum Woonzorg Brussel, in particular thanks to the advice of Katleen Deruytter. Furthermore, this application will be used as a starting point for the development of a more generic framework, in the context of a graduation thesis under the guidance of professor Kim Mens, destined to create an application framework for managing different kinds of organisations of the same type. This entire project and its possible extensions is available under an AGPv3 license, of which the integral text can be found here : https://www.gnu.org/licenses/agpl-3.0.html. The source code can be found here : https://github.com/MaximeBiset/care4care.  Contact maxime.biset@student.uclouvain.be if any problem.
